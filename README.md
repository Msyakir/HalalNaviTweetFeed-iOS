# HalalNaviTweetFeed-iOS

#### Using XCode 7.3.1, Swift 2.2, Cocoapods 1.1.1                                     

#### Libraries:                                       
* Alamofire (3.5.1)
* SwiftyJON (2.4.0)
* Nuke (3.2)
* Realm (2.0.3)

#### Features:
* Offline view
* Pull to refresh
* Load more Tweets 
* View Photo details
